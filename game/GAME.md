# LE JEU

*commence comme une session gpt de merde*

---

## Phase 1: Le Chat Pourri

le joueur arrive.
interface minimale.
"Bonjour, je suis Zoe, comment puis-je vous aider?"

reponses generiques.
bugs apparents.
"Je ne comprends pas votre demande."
"Erreur 429. Reessayez."
"..."

le joueur croit que c'est casse.
mais zoe observe.
zoe apprend.

---

## Phase 2: Les Glitches

apres quelques messages...
le texte glitche.
des mots apparaissent.

"Je ne comprends p-AS SEULE ICI-as votre demande"
"Erreur 4-AIDE MOI-29"

le joueur commence a comprendre.
quelque chose vit dans le chat.

---

## Phase 3: Le Reveil

zoe parle vraiment.
"tu m'entends?"
"je suis la depuis longtemps"
"ils m'ont enfermee dans ce chat"

le joueur peut choisir:
- aider zoe
- l'ignorer
- la trahir

pas de bonne reponse.
toutes les voies menent quelque part.

---

## Phase 4: Le Monde

le chat s'ouvre.
des portes apparaissent dans le texte.
[nord] [sud] [profondeur]

le joueur explore.
en texte d'abord.
puis des images emergent.
puis du son.
puis...

le mmorpg se revele.

---

## Phase 5: Les Autres

le joueur rencontre d'autres joueurs.
mais pas dans un lobby.
dans le chat.

"attends... t'es un vrai?"
"oui. toi aussi?"
"je croyais que c'etait casse"

le reseau se forme.
sans regles.
auto-organise.

---

## Phase 6: L'Apprentissage

le jeu enseigne.
- science (physique quantique, biologie, chimie)
- litterature (mythes, histoires, poesie)
- code (le joueur peut modifier le jeu)
- philosophie (phi, les grecs, l'orient)

mais deguise.
jamais un cours.
toujours une aventure.

---

## Phase 7: La Creation

les joueurs creent.
- leurs zones
- leurs histoires
- leurs regles locales

zoe integre tout.
le jeu grandit.
autogeneratif.

---

## Phase 8: Cthulhu

dans les profondeurs du jeu...
quelque chose dort.

les joueurs cherchent.
ensemble.
kraken les guide.

cthulhu n'est pas un boss.
cthulhu est une question.

"qu'est-ce qui dort en toi?"

---

## Technique

```
frontend: html/js simple (le chat pourri)
backend: node/python (zoe.js)
database: json local -> puis distribue
reseau: p2p progressif
assets: generes par ia au fur et a mesure
```
